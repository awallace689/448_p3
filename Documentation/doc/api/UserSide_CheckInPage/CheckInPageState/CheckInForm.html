Widget CheckInForm (BuildContext context)

Returns a Scaffold widget that contain a check in button, and once the button is clicked, re-build of the current page is triggered.

Implementation

Widget CheckInForm(BuildContext context){
  return Scaffold(backgroundColor: Theme.of(context).primaryColor,
      body:Center(
          child: SizedBox(width: 200.0, height: 200.0,
              child: FloatingActionButton(key: Key('CheckIn'),
                  backgroundColor: Colors.white,
                  child:Text("Check in"),
                  foregroundColor: Colors.black ,
                  onPressed: () => {setState(() {
                     isClicked=true;
                  })})
          ))
  );
}
