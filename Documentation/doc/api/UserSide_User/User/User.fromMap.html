User.fromMap constructor

User.fromMap(Map data)

Create User from a Map Firebase. Member variables follow DB naming scheme.

This is done using an initializer list (:), which executes before the constructor runs so you don't have to write 'this'. User user = User.fromFB(Firebase.reference().query('get user'));
