isValidPassword method

bool isValidPassword (String input)

Validate the password given.

Using regex, it validates the password give. Length of 4-8. Must include number. Must include lowercase letter. Must include uppercase letter.

Implementation

bool isValidPassword(String input)
{
  @visibleForTesting
  final RegExp regex = new RegExp(r"^(?=.*\d)(?=.*[a-z])(?=.*[A-Z])(?!.*\s).{4,8}$");
  return regex.hasMatch(input);
}
